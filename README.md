# Simple Grade Calculator

This is a small website created for a manual testing assignment.

The program allows users to enter a student name and score, then calculate the student's grade.

Functions:
- Enter student name
- Enter score
- Calculate grade
- Clear input

Some bugs were intentionally added to the program so testers can practice finding defects.

How to run the program:

1. Download the project
2. Open index.html in a web browser

Technologies used:
HTML
CSS
JavaScript
